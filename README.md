# Mirror-Executor
The official GitHub page for Mirror! It's a keyless, clean executor featuring some stuff that's gonna get added cuz there isn't much stuff yet lmao. This is the only way to download the official version of Mirror, please don't get it from anywhere else! 

The latest version of Mirror can be found under Releases as "MirrorLatest".

It will come with a folder that includes two exes, one for the main executor and another to download the latest version

You may notice that a lot of files in the folder are set to hidden. This is not to hide malware, but rather to prevent accidentally deleting essential components for the executor

Mirror will get frequent updates; that doesn't mean any big changes, but there might be big fixes and improvements.

It is recommended to run the updater every time you start up your computer, to ensure you're always on the latest version. If you don't, make sure to run it at least every 7 days.

No, old Mirror versions will not be saved to GitHub.

When running the updater, please first ensure all previous version folders of Mirror are in your Downloads folder. The system scans for old versions and deletes them, but it only checks your Downloads folder. If you don't move them into there, you'll end up with a lot of old folders.

The folder name will never change, it will always be "MirrorLatest". You can verify whether the actual executor as been updated by either checking the name of the exe (e.g. Mirror-X.X.X) or opening the Changelog txt, which will show you when the update come out, which you can then compare to the current date to see if you're on the latest version.
